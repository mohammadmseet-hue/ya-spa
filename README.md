# يا سبا · Ya Spa

At-home beauty & spa marketplace for Saudi women — **verified, licensed, women-only**.
Arabic-first (RTL), fully bilingual (AR / EN), light-pink premium brand. Static site, no build step.

🔗 **Live:** https://mohammadmseet-hue.github.io/ya-spa/

## Pages
- `index.html` — landing page (hero, services, live price estimator, trust stack, bridal, Ya Spa Plus, safety, FAQ, waitlist)
- `deck.html` — investor pitch deck (← → / swipe to navigate)
- `compliance.html` — KSA licensing & compliance checklist (progress saved locally)

## Stack
Hand-written HTML / CSS / vanilla JS. Google Fonts (Tajawal, IBM Plex Sans Arabic/Latin, Fraunces).
Design tokens, WCAG-checked pink palette, logical CSS properties for RTL/LTR.
